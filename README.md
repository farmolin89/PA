# PA

Корпоративный портал

## Запуск сервера

1. Убедитесь, что установлен Node.js версии 16 или выше и пакетный менеджер npm 8+.
2. Склонируйте репозиторий и перейдите в его корневую директорию.
3. Выполните установку зависимостей командой `npm install` (или `npm run setup`).
4. Для запуска в режиме разработки воспользуйтесь командой `npm run dev`.
   - Скрипт автоматически установит зависимости, если папка `node_modules` отсутствует.
   - Если локальная копия `nodemon` недоступна, будет предпринята попытка запуска через `npm exec nodemon`.
   - При недоступности nodemon сервер всё равно запустится, но без автоматической перезагрузки.
5. Для запуска в обычном режиме используйте `npm start`.

После успешного запуска сервер будет доступен по адресу `http://localhost:3000/`.
